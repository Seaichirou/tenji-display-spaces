# 出典と備考

**出典**  
オリジナルの設問です。

**原文**
　つぎの　1〜6で　つかわれる　「カイソー」を　
ようれい　a~fと　すべて　ただしく　むすびつけてください。
1、海藻が取れる海域だ。
2、高い階層わ眺めがいい。
3、あの頃をよく回想する。
4、店わ改装中だ。
5、回送中のタクシーにわ乗れない。
6、この馬の走りわまさに快走だな。
a.回送電車。
b.回想録。
c.海藻サラダ。
d.改装工事。
e選手の快走が楽しみ。
f.社会階層。
　つぎの　あ〜か　の「カイソー」と、1〜6の　れいぶんとを　
すべて　ただしく　むすびつけてください。　
あ、快走。
い、海藻。
う、階層。
え、回送。
お、改装。
か、回想。
